# Landing Page - Divulgação da Solução

## Descrição
Este projeto consiste em uma landing page estática para divulgação de uma solução inovadora. A página é projetada com animações, flex-box para layout, formulário de contato e é totalmente responsiva para oferecer uma boa experiência em diferentes dispositivos.

## Funcionalidades
- **Animação**: A página inicial apresenta uma animação de entrada no cabeçalho.
- **Layout Flexível**: Uso de flex-box para um layout moderno e responsivo.
- **Formulário de Contato**: Um formulário simples para coleta de informações de contato.
- **Responsividade**: Layout ajustável para diferentes tamanhos de tela, garantindo boa usabilidade em dispositivos móveis e desktops.

## Tecnologias Utilizadas
- **HTML5**: Estrutura da página.
- **CSS3**: Estilização e responsividade.
- **JavaScript**: Funcionalidade do formulário.

## Estrutura de Arquivos
- `index.html`: Arquivo principal com a estrutura HTML da página.
- `styles.css`: Arquivo de estilos CSS para layout e animações.
- `script.js`: Arquivo JavaScript para funcionalidades interativas do formulário.
